The bottom sheet is a versatile UI component that allows you to render any content within a temporary panel anchored to the bottom of the screen. It can be used for various purposes such as displaying actions, collecting user input, or showing additional information.

You can pass any standalone component to render inside this bottom sheet. The component you provide should manage its own styles such as padding, margin, and layout, as the bottom sheet is designed to be flexible and generic.

For advanced usage, you can customize the header or disable the close button. This allows for creating a custom header with personalized actions, like using a custom close button to dismiss the bottom sheet.
